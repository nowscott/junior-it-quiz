# 初中信息技术选择题练习系统

这是一个为初中生设计的在线信息技术选择题练习与考试系统。该项目旨在帮助学生通过模块化练习和模拟考试，巩固信息技术基础知识。

## 项目特点

- **模块化练习**：涵盖信息技术基础、Windows 操作系统、网络基础、WPS 文字、WPS 表格、WPS 演示文稿等六大核心模块。
- **随机模拟考试**：支持从题库中随机抽取 20 道综合题目进行考试，自动计时（60分钟），支持提交后查看评分和详细错题分析。
- **题目精确定位**：支持通过题号快速跳转，方便学生针对特定题目进行复习。
- **详细答案解析**：每道题目作答后即刻显示详细解析，帮助学生理解知识点。
- **图文互动**：支持题目插图显示，并提供点击放大查看功能，提升答题体验。
- **零依赖运行**：纯前端开发，无需任何后端环境或数据库，本地双击即可运行。

## 目录结构

项目已完成模块化重构，文件排布整洁有序：

- [index.html](index.html)：系统入口主页面。
- **assets/**：静态资源核心目录。
  - **css/**
    - [style.css](assets/css/style.css)：全局样式表，负责页面布局与响应式设计。
  - **js/**
    - [data.js](assets/js/data.js)：题目数据库，存储所有模块的题目、选项、答案及解析。
    - [app.js](assets/js/app.js)：核心逻辑脚本，处理答题、考试计时、跳转及 UI 更新。
  - **images/**：存放所有题目配套的插图文件。

## 使用指南

1. **开始练习**：直接在浏览器中打开根目录下的 `index.html`。
2. **切换模块**：通过左侧侧边栏选择感兴趣的知识模块。
3. **快速跳转**：在右侧练习区域上方输入题号并点击“跳转”即可定位题目。
4. **进行考试**：点击侧边栏最下方的“随机综合考试”，确认后开始计时答题。
5. **查看解析**：在练习模式下，点击选项后解析区域会自动展开。

## 维护与扩展

- **题目更新**：如需新增或修改题目，请直接编辑 `assets/js/data.js` 中的 `questionData` 对象。
- **样式修改**：如需调整 UI 界面，请编辑 `assets/css/style.css`。
- **功能迭代**：核心业务逻辑位于 `assets/js/app.js`，可在此处扩展新的功能模块。
